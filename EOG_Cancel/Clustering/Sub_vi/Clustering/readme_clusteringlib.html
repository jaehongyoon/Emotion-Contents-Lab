Clustering Library Readme

This file contains information to introduce you to the Clustering Library. Please note that this library is not formally released, and is therefore not supported by National Instruments. This library is a set of VIs and demos designed as starting points for clustering analysis in LabVIEW.

Overview

The goal of clustering is to partition data samples into different classes according to the samples' mutual similarity. First, a distance metric is defined to measure similarity. Next, by minimizing the overall cost, you can determine the centroids and partition scheme. This process is often iterative and the cluster centroids will converge after several iterations. Possible applications of clustering include pattern recognition, data compression, image segmentation, and so on.

System Requirements

To use the Clustering Library, you must have LabVIEW 8.5 or later installed on the computer.

Installation Instructions

Complete the following steps to install the Clustering Library:

  1. Verify that LabVIEW is not running.
  2. Run the setup.exe file.
  3. Follow the instructions that appear on the screen.

The Clustering VIs appear in the Addons category on the Functions palette in LabVIEW.

Uninstallation Instructions

Complete the following steps to uninstall the Clustering Library.

  1. Open the Add or Remove Programs dialog box from the Control Panel.
  2. Select National Instruments Software and click the Change button.
  3. Under Product, select NI Labs Clustering Library and click the Remove button.

Getting Started

Open and run the demos located in the labview\vi.lib\addons\Clustering\Demo directory. These demos also are available by navigating to Start»All Programs»National Instruments»NI Labs»Clustering Library»Demo. The demos in this directory demonstrate two-dimension samples clustering and image segmentation application.

The front panel of each demo displays a description of the demo and instructions on how to use the demo.

Documentation

The clustering library implements the following four methods of clustering:

  • K-means and K-medoid Algorithms
  • Fuzzy C-means Algorithm
  • Gaussian Mixture Model
  • Hierarchical Clustering

Refer to the NI_Clustering_Concept.pdf file located in the National Instruments\NI Labs\Clustering Library directory for more information about these algorithms.
